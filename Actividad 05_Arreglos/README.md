# Arreglos
## Descripción
Programas en C que practican el uso de arreglos para
almacenar y manipular colecciones de datos del mismo tipo.

## Temas aplicados

- Declaración e inicialización de arreglos
- Acceso a elementos de un arreglo por índice
- Arreglos de tamaño dinámico, ingresador por el usuario
- Recorrido de arreglos con ciclo `for`
- Direcciones de memoria con `&`
- Arreglos de tipo `float`

## Ejercicios

- **Codigo 1:** Exploración de direcciones de memoria de variables y arreglos, utilizando el operador `&`
- y con %i para indicar que el dato es un número entero con signo,  acceso directo a elementos del arreglo con sus indices
e ingreso de 4 temperaturas con un ciclo for, y con impresión de resultados
- **Codigo 2:** Arreglo de tamaño N definido por el usuario para almacenar
  e imprimir temperaturas ingresadas dinámicamente

## Compilación
```bash
gcc main.c -o programa
```

## Ejecución
```bash
./programa
```
