

# Actividad: Tipos de datos en C
## Descripción
Programa en C que imprime en terminal todos los tipos de datos, sus valores y direcciones de memoria. 
## Temas aplicados
-Tipos de datos primitivos en C
-Valor en bytes de los tipos de datos
-Uso de & para obtener direcciones de memoria
## Compilación
```bash
gcc main.c -o programa
```
## Ejecución
```bash
./programa
```
